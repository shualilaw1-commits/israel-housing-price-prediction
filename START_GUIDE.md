## 📋 שלבים להפעלה

### שלב 1: וודא שהסביבה מוכנה
```bash
# ודא שכל החבילות מותקנות
pip install -r requirements.txt
```

### שלב 2: בחר מה להריץ

יש לך 3 אפשרויות:

#### אפשרות 1: הרצת Dashboard בלבד (מומלץ להתחלה)
```bash
# Windows - לחץ פעמיים על:
1_RUN_DASHBOARD.bat

# או הרץ ידנית:
streamlit run app/streamlit_app.py
```

#### אפשרות 2: אימון מודל חדש
```bash
# Windows - לחץ פעמיים על:
2_TRAIN_MODEL.bat

# או הרץ ידנית:
python train_model_manually.py
```

#### אפשרות 3: הרצת הפרויקט המלא (CrewAI)
```bash
# Windows - לחץ פעמיים על:
3_RUN_FULL_PROJECT.bat

# או הרץ ידנית:
python run.py
```

## 📊 מה כל קובץ עושה?

### קבצי BAT (Windows)
- **1_RUN_DASHBOARD.bat** - פותח את הממשק האינטראקטיבי
- **2_TRAIN_MODEL.bat** - מאמן מודלים חדשים
- **3_RUN_FULL_PROJECT.bat** - מריץ את כל הפרויקט עם CrewAI

### קבצי Python ראשיים
- **app/streamlit_app.py** - הממשק האינטראקטיבי
- **train_model_manually.py** - אימון מודלים ידני
- **run.py** - הרצת CrewAI Flow המלא
- **test_prediction.py** - בדיקת חיזויים

### תיקיות חשובות
- **outputs/** - כל הקבצים שנוצרים (מודלים, נתונים, דוחות)
- **crews/** - הגדרות ה-Agents של CrewAI
- **app/** - קבצי הממשק

## 📝 מה יש בפרויקט?

### נתונים קיימים ב-outputs:
- ✅ `clean_data.csv` - נתונים מנוקים
- ✅ `features.csv` - פיצ'רים מהונדסים
- ✅ `model.pkl` - מודל מאומן
- ✅ `dataset_contract.json` - תיאור הנתונים
- ✅ `all_models_comparison.json` - השוואת מודלים
- ✅ `city_mapping.json` - מיפוי ערים

### מה חסר?
אם חסרים קבצים נוספים (insights, evaluation, model_card), הרץ:
```bash
python run.py
```
